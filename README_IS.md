# GKI2026

GKI2026 er safn af litlum AI/ML keppnisverkefnum fyrir keppnina. Hvert undirmappa inniheldur sjálfstætt verkefni, dæmi um líkanakóða og lágmarks skriftur til að keyra eða staðfesta innsendingar.

Uppbygging í trjám (efst):
- `golden_plate_on_thingvellir_NLP/` — NLP verkefni: spá um næsta bæti (gögn, þjálfun, innsendingartól)
- `habitat_classification/` — myndaflokkunarverkefni með líkani og hjálparföllum
- `hierarchical_time_series_forecasting/` — tímaraðarspá verkefni
- `submission/` eða verkefnissértækar `submission/` möppur — dæmi um `model.py` og innsendingarmynstur

Hraður byrjun: opnaðu verkefnismöppu, lesið `README.md` þar og fylgið leiðbeiningum (sækja gögn, þjálfa, `create_submission.py` og `check_submission.py`).
